## 🧠 About the Project

The **Investlite Demo Platform** is a front end demonstration of an investment service platform, built to give developers a foundation for:

🔥 Learning modern web development tools
💡 Testing investment-based UI flows
📌 Integrating with backend services (like Supabase)
👨‍💻 Customizing frontend logic

This README provides clear instructions and context so contributors and users can get started quickly and understand the purpose of each part of the code.

## 🚀 Getting Started

Here’s how to set up this project locally.

### 📥 Prerequisites

Make sure you have the following installed:
✔️ Node.js (v16+)
✔ npm or yarn
✔ Git

### 💻 Install

1. **Clone the repository**

   ```sh
   git clone https://github.com/ade-gb/investlite-demo-platform.git
   ```

2. **Go into the project directory**

   ```sh
   cd investlite-demo-platform
   ```

3. **Install dependencies**

   ```sh
   npm install
   ```

---

## 🧪 Running the App

Start the development server with:

```sh
npm run dev
```

Then open your browser and visit:

```
http://localhost:3000
```

You should see the investment demo platform running with hot-reload support for fast changes.

## 📁 Project Structure

```plaintext

│   ├── components/     # UI components
│   ├── pages/          # Top level app pages
│   ├── styles/         # Global styles
│   └── utils/          # Utility functions

├── tsconfig.json       # TypeScript config
├── tailwind.config.ts  # Tailwind setup
└── vite.config.ts      # Vite config
```

This structure keeps UI, logic, and config cleanly separated.
